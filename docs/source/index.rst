Welcome to KUCC-Discord-Bot's documentation!
============================================

.. note::

   This project is under active development.

**KUCC-Discord-Bot** is a internally devloped bot written in Python language to handle 
the need of moderating the official **Kathmandu University Computer Club (KUCC)** Discord Server.

It is used to verify KUCC members and alot of other acts.


Check out the :doc:`usage` section for further information, including how to
:ref:`install <installation>` the project and :doc:`Server Proposal` to get information
about the planned Discord Server.
